# miniweb-grup-GJS
Aquest projecte és una miniweb feta en equip pels alumnes de 2n de DAW per practicar el treball col·laboratiu amb GitHub.
La web esta composta de diferents apartats, tenim una secció de benvinguda, una presentació de la web i finalment uns enllaços amb recursos, incloent header i footer.

## breu resum del flux utiltizat
1. Clona el repositori:

git clone https://github.com/gerard-bardeli/miniweb-grup-GJS.git

2. Crea una branca amb el teu nom:
 
git checkout -b nom

3. Fes els teus canvis i puja’ls:
 
git add .
git commit -m "Afegeixo part X"
git push origin nom

4. Crea un Pull Request a GitHub per fusionar els canvis.
